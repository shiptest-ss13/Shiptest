05 March 2022

MarkSuckerberg updated:

  • Empty space docks now have four docking ports instead of two.
  • You can't just spawn in overmap objects like you used to be able to anymore.
  • Adds new verb, `View overmap` to view the entire overmap in a really basic HTML window.
  • You no longer have to set launch_status on future ship map docking ports.
  • Removed autopilot console.
  • Mapvotes. They didn't work anyways.
  • Integrity. This was a bad system, it can be re-implemented in another PR in a much better way.
  • Custom shuttles work again.
  • Ruin-loaded shuttles work again.
  • Pretty much all of overmap code, for as smooth an experience yet.
  • The runtime viewer shouldn't hang the server for like 30 seconds anymore.
